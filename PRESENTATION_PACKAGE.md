# Paket Presentasi e-Kaku V3

## Lokasi Script
- `scripts/presentation/setup-presentasi.bat`
- `scripts/presentation/jalankan-server.bat`
- `scripts/presentation/cek-cepat.bat`

## Cara Pakai (Windows)
1. Buka terminal di root project.
2. Jalankan setup penuh:
```bat
scripts\presentation\setup-presentasi.bat
```
3. Jalankan server:
```bat
scripts\presentation\jalankan-server.bat
```

## Cek Cepat Sebelum Presentasi
```bat
scripts\presentation\cek-cepat.bat
```
